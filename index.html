All the Cats, All the Pictures

Free Code Camp Image Search Abstraction Layer

This API tool allows you to submit an image search query and view the results as a JSON object.

You may also specify the offset (where your results should begin), and view recent search queries.

Usage:

This submits a search for "fashionable cats" with an offset of 5: https://custom-image-search.herokuapp.com/api?find= fashionable cats &offset= 5

Alternatively, you may view recent searches this way: https://custom-image-search.herokuapp.com/recent

Any improperly formated queries should redirect you to this page.
