Free Code Camp Image Search Abstraction Layer

This API tool allows you to submit an image search query and view the results as a JSON object.

You may also specify the offset (where your results should begin), and view recent search queries.

Usage:

Search for "fashionable cats" with an offset of 5: https://custom-image-search.herokuapp.com/api?find=fashionable%20cats&offset=5

View recent searches this way: https://custom-image-search.herokuapp.com/recent

You may include spaces in your search and specifying the offset is optional (the default is 1). Any invalid queries return here. Now search for all the cat pictures.

How does this project works

Built by Sean Smith

This tool was built with Node.js Express.js, MongoDB and Google's Custom Search Engine API.

Search queries are parsed and then a request is made to Google's servers, which returns a JSON object. Certain elements from this response are then returned to the user.
(Note: I would recommended using a JSON parser Chrome plugin like this one to view the page output.)

Each user query also gets stored in a MongoDB database with a timestamp recording when it is executed, which allows users to view recent searches.

The application is hosted with Heroku, and the database is hosted by mLab.
